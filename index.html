🌿 Bienvenido a AgroTech Invest

Simulador de análisis de inversión para tecnología agroalimentaria

1

⚙ Configuración — empezá acá

Seleccioná el país (el tipo de cambio se carga automáticamente). Ingresá los litros por día, los días operativos por mes y el precio de venta que esperás obtener por litro. Completá la inversión inicial desglosada.

✓ Los 4 indicadores del fondo se actualizan automáticamente
2

💰 Costos — ajustá los valores reales

Los valores por defecto son referencias internacionales. Ajustá al menos: leche cruda (= precio que pagás al productor), sachet, energía, mano de obra y alquiler. El resto podés dejarlo y ajustar después.

✓ España: el impuesto plástico UE ya está incluido
3

📊 Análisis Micro — ¿es rentable el nodo?

Muestra el resultado mensual, el punto de equilibrio (cuántos litros necesitás vender para no perder) y la comparativa vs. vender la leche sin procesar.

✓ Si el resultado es negativo, probá aumentar el precio o el volumen
4

📈 Flujo de Caja — VAN, TIR y Payback

El gráfico de 36 meses muestra cuándo recuperás la inversión. La tabla anualizada incluye el flujo descontado. La curva VAN vs. tasa te muestra la robustez del proyecto.

✓ VAN positivo + TIR mayor a la tasa = proyecto viable
5

🎯 Escenarios — stress test del proyecto

La tabla de sensibilidad 5×5 muestra el VAN para distintas combinaciones de precio y volumen. El simulador con sliders permite explorar cualquier combinación en tiempo real.

✓ Celdas verdes = proyecto viable en ese escenario
6

🌐 Red de Nodos — análisis de escala

Agregá múltiples nodos con precios y costos distintos (distintas regiones). La proyección temporal muestra la expansión a 10 años con 3 modelos de crecimiento.

✓ Los nodos se pueden renombrar y editar
7

🌿 Huella Ambiental — fondos verdes

Calcula la huella de carbono y hídrica con benchmarks científicos. Determina automáticamente la elegibilidad para fondos FEADER, LIFE, CDTI y banca ética.

✓ Valores por defecto = benchmarks validados para pequeña escala europea
8

📋 Reporte — exportar el análisis

Genera un informe completo con score financiero (0-100) + score ESG, conclusión automática y brechas críticas. Ctrl+P → Guardar como PDF para exportar.

✓ Usar 💾 para guardar el proyecto como JSON y 📂 para cargarlo después
